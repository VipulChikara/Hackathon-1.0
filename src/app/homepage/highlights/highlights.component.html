What is Helping Hands?

Helping Hands is a module based platform for creating platforms which that provides a helping hand for any donor-beneficiaries system.

We aim to bridge the gap between donors and beneficiaries, so that they can focus on important things and contribute together for a better, sustainable and healthy India.
